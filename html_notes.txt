1. Basic HTML Structure
- <!DOCTYPE html> - Declares the document type and HTML version.
- <html> - Root element of an HTML document.
- <head> - Contains meta-information about the document.
- <title> - Sets the title of the document (shown in browser tab).
- <body> - Contains the content of the HTML document.

2. Head Elements
- <meta> - Provides metadata like charset, author, etc.
- <link> - Links external resources like CSS.
- <style> - Embeds internal CSS.
- <script> - Embeds or references JavaScript.
- <base> - Sets the base URL for all relative URLs in the document.

3. Text Content
- <h1> to <h6> - Headings from largest to smallest.
- <p> - Paragraph.
- <br> - Line break.
- <hr> - Thematic break (horizontal line).
- <blockquote> - Section for quoted text.
- <pre> - Preformatted text.

4. Inline Text Semantics
- <b> - Bold text.
- <strong> - Important text (bold).
- <i> - Italic text.
- <em> - Emphasized text (italic).
- <mark> - Highlighted text.
- <small> - Smaller text.
- <del> - Deleted text.
- <ins> - Inserted text.
- <sub> - Subscript.
- <sup> - Superscript.
- <span> - Generic inline container.

5. Grouping Content
- <div> - Generic container for flow content.
- <ul> - Unordered list.
- <ol> - Ordered list.
- <li> - List item.
- <dl> - Description list.
- <dt> - Description term.
- <dd> - Description details.
- <figure> - Self-contained content, often with caption.
- <figcaption> - Caption for the <figure> element.

6. Links and Media
- <a> - Anchor (hyperlink).
- <img> - Image.
- <audio> - Audio content.
- <video> - Video content.
- <source> - Media resource.
- <track> - Text tracks for media.
- <iframe> - Inline frame.

7. Table Elements
- <table> - Table container.
- <caption> - Table caption.
- <thead> - Group of header rows.
- <tbody> - Group of body rows.
- <tfoot> - Group of footer rows.
- <tr> - Table row.
- <th> - Table header cell.
- <td> - Table data cell.

8. Forms and Input
- <form> - Form container.
- <input> - Input field.
- <textarea> - Multi-line text input.
- <button> - Clickable button.
- <select> - Drop-down list.
- <option> - Option in a drop-down.
- <label> - Label for input.
- <fieldset> - Groups related inputs.
- <legend> - Caption for <fieldset>.
- <datalist> - Predefined options for input.
- <output> - Result of a calculation.

9. Semantic Elements
- <header> - Header section.
- <nav> - Navigation links.
- <main> - Main content area.
- <section> - Thematic grouping of content.
- <article> - Self-contained content.
- <aside> - Sidebar content.
- <footer> - Footer section.
- <address> - Contact information.
